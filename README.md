### 📚 What I learned

During this project, I practiced responsive design and grid layouts.

Here’s a small code snippet I’m proud of:

```css
.row {
  display: grid;
  grid-template-columns: 1fr 1fr;
}

@media (max-width: 600px) {
  .row {
    grid-template-columns: 1fr;
  }
}
```
